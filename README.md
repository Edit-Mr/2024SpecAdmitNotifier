## 如何使用

1. **留下一顆星星**。你懂的。
2. **Fork 這個專案**，右上角那個。
3. **設定 Secrets**
   
   在你的專案頁面點選 `Settings` -> `Secrets` -> `actions` -> `Repository secrets`，新增 Secrets：
   - `WEBHOOK_URL`: Discord Webhook URL。請到你的 Discord 伺服器設定 --> 整合 --> Webhook --> 新 Webhook，複製 Webhook URL。
